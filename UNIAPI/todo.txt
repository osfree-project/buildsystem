x. add default modifier to uses

uses os2 default;

will produce

#define INCL_OS2
#ifdef INCL_OS2
#include <os2.h>
#endif

x. add alias modifier to uses

uses bse alias base;

will produce

#ifdef INCL_BASE
#define INCL_BSE
#endif
#ifdef INCL_BSE
#include <bse.h>
#endif

x. error key to define error

error [ERROR_ID] error_number MESAGE_ID 'message';

ERROR_ID is optional

As result must be produced:

error ERROR_PATH_NOT_FOUND 123 MSG_PATH_NOT_FOUND 'Searched path not found' 'oso001.txt';

#define ERROR_PATH_NOT_FOUND 123
#define MSG_PATH_NOT_FOUND   123

And in oso001.txt this string will be added

SYS0123I: Searched path not found
